# 매일 다이어리

## 업데이트 안내

이 저장소는 **다이어리 앱의 업데이트 알림**용으로 사용됩니다.

### 새 버전이 나왔을 때
1. 다이어리 앱을 켜면 상단에 **"🔔 업데이트 있음"** 배너가 표시됩니다.
2. 배너를 클릭하면 이 페이지가 열립니다.
3. **새 버전 exe 파일**은 사내메일 또는 카톡으로 별도 전달됩니다.
   - (사내 방화벽 정책상 GitHub에 큰 파일을 올릴 수 없어 직접 전달합니다.)
4. 전달받은 `다이어리.exe` 파일로 기존 파일을 **덮어쓰기**하면 업데이트 완료.

### 현재 버전 확인
이 페이지 상단의 최신 version.json 값을 확인하세요.

### 데이터 보존
업데이트(파일 교체)를 해도 기존에 쓰던 다이어리 내용은 그대로 유지됩니다.
(데이터는 브라우저에 저장되어 앱 파일과 분리되어 있습니다.)
